# Control System Modelling 
Repository contains Jupyter files with control system modelling of a gold elution column. <br>
This forms part of the final year Process Control module (INGC 413).
<br>
**sympy_linearisation.ipynb** contains the complete derivation of the transfer function models with the expressions for gains and time constants. <br>
**steady_state_solutions.ipynb** contains the code for determination of steady-state values and numerical values of the gains and time constants used in the transfer function models.
**stability_analysis.ipynb** contains symbolic math to obtain expressions for frequency response analysis.
